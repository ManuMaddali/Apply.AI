# Usage Instructions

1. **For Basic Testing**: Use the URLs listed under each profession with the corresponding test resume
2. **For Cross-Industry Testing**: Use URLs from different categories to test transferable skills
3. **For Experience Level Testing**: Apply entry-level resumes to senior roles and vice versa
4. **For RAG Testing**: Submit multiple URLs to build historical data, then test with similar roles

- These are placeholder URLs - replace with actual job postings for live testing

- Test with a mix of company sizes (startup, mid-size, enterprise)
